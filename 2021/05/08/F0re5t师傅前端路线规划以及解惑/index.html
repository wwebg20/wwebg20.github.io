F0re5t师傅前端路线规划以及解惑

交流学习（x) 传道解惑（√）

正文开始

  • Q1:新手前端学习路线
Css html JavaScript

  1. HTML 主要是四个重要标签，其他的开发过程过不是特别常用

    Head / body /div / span

  2. CSS:

    “你看了很多，不用，都会忘”

    • Position：子绝父相定位

    • Display

    • 盒模型

    • Flex 布局：能够解决95%的布局问题：http://caibaojian.com/flexbox-guide.html

    • Float 浮动（脱离文档流，塌陷问题）

  3. JavaScript

    • 红宝书（第三版）第三章 ～ 第七章，建议看2～3次!

      • 第三章 基本概念
      • 第四章 变量、作用域和…
      • 第五章 引用类型
      • 第六章 面向对象（对就是我没有的这种东西）的程序…
      • 第七章 函数表达式

    • 红宝书看无聊去看看 你不知道的js，这本黄皮书，第一本好好看看(上册)

    • 学了 js ，es6（箭头函数，let/const，async，await，promise），《深入理解ES6》

    ​

  4. Vue / react 2选1

    ​ Js 学的好的话，选react

    ​ Js 基础不太好，选 vue

  • Q2： 前端项目有哪些是具有含金量的，值得推荐的？
    “不要为了做项目而做项目”

    ​ 不要盲目地从github上下载一个项目就开始去做，要带有目的性，自己学习去做项目是可以的但 是.写在简历上就显得很水

  • Q3 : 有哪些认为有必要购买纸质的书籍么

    • 红宝书

    • 你不知道的js 上册

    • 深入理解ES6

    • 入门：vue 实战

    • 深入： 深入浅出 vue（前三章）

    • React：官方文档

    • webpack：webpack 入门，进阶，调优

    • 设计模式：JavaScript设计模式（alloyteam出版）

  • Q4:三件套应该怎么学呢，是看视频跟着 pink 老师还是怎么学，来避免学习效率低
    • pink老师讲的不错的
    • 不要单一，如果你感觉视频慢了就去看文档，有一些文档就是最好的教程（react官网）
  • Q5:学校有开一些硬件的课，比如数电以及实验什么的，有必要学的很透彻么，这些和基础知识(如计网计组)关联大么
  • Q6:如果想去中厂，操作系统 ，数据结构算法，计网，哪个需要花比较多的功夫呢

    A5、6 计网最重要！！

  • Q7:大二怎么投实习，需要内推吗，还是自己海投就行。

    • 牛客网 一堆人提供内推

    • 实习僧

    • Boss 直聘

    • 官网投递

    • 非官网记得都要慎重，可能都是外包的啊或者那种劳务工

  • Q8:请问大佬怎么决定自己的技术方向的呢

    • Serverless Faas

    • 可视化 echarts d3

    • 前端富文本编辑器（飞书文档，腾讯文档）

  • Q9：大佬对三十五岁危机有什么看法 对以后人生的规划
    “我只是个年轻人啊我能有什么看法！”
  • Q10:大佬从小白到找一个实习至少要学多长时间
    半年时间，每天五到六小时，字节跳动欢迎你
  • Q11:博客可以用别的平台代替么 比如知乎专栏啊公众号啊简书掘金什么的
  • Q12:课内的各种数学，高数线代离散，只保证不挂科可以么

    ​ A11、12 : “当然可以”

  • Q13:三大件学完之后首先学什么？jquery.ajax等不用学嘛？
    • Jquert 学它的优秀思想
    • Ajax 就是一种技术的体现而已，不是jquery的专利
  • Q14:vue应该学到什么程度
    • 使用 vue 全家桶做一个项目，具有比较大的复杂度
  • Q15:基础课程都有哪些是必须的，以及顺序（提前自学的那种）
    • 计算机啊网络
    • 数据结构
    • 操作系统
    • 数据库原理(对于纯前端来说不太重要)
  • Q16：有能力的话可以成为全栈工程师吗
    • 看各个厂要求（全栈当然好啦）
